Week 9 Recap

Graphics

In week 9 we looked at the available graphics systems in R. While there are four of them we will concentrate only on two of them: Base Graphics and Lattice Graphics. These are the two most popularly referenced packages when building plots with R. However, ggplot is a very nice package and warrants attention - just not in this class.

In terms of Base graphics we learned that it contains both high and low level functions which makes it easy to create a plot using a specific function (e.g. hist, plot, barplot) or by "building" a layered plot using primitive functions such as points, lines, and abline. Regardless of approach it is also easy to annotate an existing plot using functions such as text, mtext, arrows, and other functions that allow us to draw on top of what is already there. For some this is a tedious way to do things although for others it is preferred since they can build their plots in stages.

We also learned about "panels" and how they relate to the basic plot window within R. Panels allow us to display relationships between continuous variable across different category values. As an example we could inspect the MPG vs Wt relationship across the three cylinder groups by using Base graphics. The following will create three separate panels each of which contains the MPG vs wt relationship for the 4,6, and 8 categories. In week 10 we will learn about the lattice package and how it implements paneling.

par(mfrow=c(1,3))

fourcyl=mtcars[mtcars$cyl==4,]

sixcyl=mtcars[mtcars$cyl==6,]

eightcyl=mtcars[mtcars$cyl==8,]

plot(fourcyl$wt,fourcyl$mpg,main="MPGvsWtfor4Cyl",ylim=c(0,40))

plot(sixcyl$wt,sixcyl$mpg,main="MPGvsWtfor6Cyl",ylim=c(0,40))

plot(eightcyl$wt,eightcyl$mpg,main="MPGvsWtfor8Cyl", ylim=c(0,40))
